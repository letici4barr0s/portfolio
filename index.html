Previsão do Tempo

Aplicação web desenvolvida para consultar previsões do tempo de forma simples e rápida, permitindo ao usuário informar uma cidade e uma data para obter temperatura máxima, temperatura mínima e probabilidade de chuva. O sistema busca automaticamente as coordenadas da cidade e consulta uma API externa de dados meteorológicos para gerar a previsão. Sistema de Gerenciamento de Pacientes e Consultas

Sistema desenvolvido para gerenciamento de pacientes e consultas médicas em ambiente de terminal, permitindo cadastrar, listar, editar e remover pacientes, além de agendar, editar e cancelar consultas. O projeto também inclui autenticação de usuários, controle de fila de atendimento e restauração de consultas canceladas, com armazenamento de dados em arquivos para manter as informações salvas.
